Usando links externos

Quando fizer uma âncora ou link externo, colocar as tags TARGET = "_BLANK e REL = "EXTERNAL" para que o link abra em outra tab e mantenha a sua no navegado.

Exemplos abaixo com target apenas para os dois primeiros.
